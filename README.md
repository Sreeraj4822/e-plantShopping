# e-plantShopping

## Project Description

e-plantShopping is a React-based plant shop application that allows users to browse plants, view details, and manage a shopping cart.

## Features

- Landing page with company name and background image
- About Us section
- Product listing page
- Shopping cart functionality
- Add item to cart
- Remove item from cart
- Update item quantity
- Total cart calculation using Redux Toolkit

## Technologies Used

- React
- Redux Toolkit
- JavaScript
- CSS
